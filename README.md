# BetterThaana Userstyles

Custom userstyles for Maldivian websites with improved Dhivehi font rendering.

## Features

- **Font Replacement**: Replace default fonts with better alternatives
  - Body text: Tharujamaanu, MV Typewriter, or Faruma (user selectable)
  - Headlines: MV A Waheed
- **Font Scaling**: Automatic size adjustment for Tharujamaanu (110%)
- **Ad Removal**: Hide ads on supported sites

## Font Options

Each userstyle allows you to choose your preferred body font:

| Font | Description |
|------|-------------|
| **Tharujamaanu** | Modern, clean design with automatic 110% size scaling |
| **MV Typewriter** | Classic typewriter aesthetic |
| **Faruma** | Traditional Dhivehi font |

Headlines use **MV A Waheed** for improved readability.
